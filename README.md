# Laundry Service Webpage  
A clean and responsive landing page for a Laundry Service, built using **HTML + CSS (no flexbox)**.  
This project includes a navigation bar, hero section, call‑to‑action button, and responsive design for mobile screens.

---

## 🚀 Features

### ✅ Navigation Bar
- Logo on the left  
- Centered menu links  
- Username button on the right  
- Fully responsive on mobile (stacked layout)

### ✅ Hero Section
- Left side: Heading, description, CTA button  
- Right side: Laundry-themed image  
- Smooth spacing and clean typography  
- Button with hover animation and shadow

### ✅ Responsive Design
- Uses `@media` queries  
- Layout shifts to single-column on mobile  
- Navbar becomes stacked for small screens  
- Images resize smoothly without distortion

---

## 🛠️ Technologies Used
- **HTML5**  
- **CSS3**  
- No Flexbox used (as per assignment requirement)  
- Responsive design using media queries
